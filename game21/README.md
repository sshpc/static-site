# 21点游戏 (Blackjack Game)

这是一个基于HTML、CSS和JavaScript开发的网页版21点游戏，具有简洁美观的界面和流畅的游戏体验。

## 游戏概述

21点是一种经典的扑克牌游戏，玩家的目标是让手中的牌点数尽可能接近21点，但不超过21点，同时击败庄家。本游戏实现了完整的21点规则，并添加了一些特色功能增强游戏体验。

## 功能特点

- **完整的21点游戏逻辑**：包括发牌、要牌、停牌、计分等核心功能
- **多样化押注系统**：支持5分、10分、20分、50分和100分的押注选项
- **积分倍数机制**：每多要一张牌，赢得的积分倍数增加0.5倍
- **爆牌惩罚系统**：爆牌时根据要牌数量倍增扣除积分
- **视觉效果增强**：包含爆牌动画、游戏结束动画和闪烁提示
- **响应式设计**：适配不同屏幕尺寸的设备
- **直观的用户界面**：清晰的牌面显示和操作按钮
- **详细的游戏说明**：帮助新玩家快速了解游戏规则

## 技术实现

- **HTML5**：构建游戏页面结构
- **CSS3**：实现样式和动画效果
- **JavaScript ES6+**：使用面向对象编程实现游戏逻辑
- **事件驱动**：通过事件监听器响应用户操作
- **DOM操作**：动态更新游戏界面元素
- **定时器管理**：实现游戏流程和动画效果的时序控制

## 游戏规则

### 基本规则

1. **游戏目标**：尽可能让牌点数接近21点，但不要超过21点
2. **点数计算**：
   - J、Q、K 各算10点
   - A 可以算1点或11点（系统自动选择最优值）
   - 其他牌按牌面数字计算
3. **游戏流程**：
   - 玩家选择押注金额
   - 点击「开始」按钮开始游戏
   - 玩家可以选择「要牌」获取一张新牌，或选择「停牌」结束回合
   - 庄家会在点数小于17时继续要牌
   - 根据双方点数决定胜负

### 特色规则

1. **积分倍数**：每要一张额外的牌，赢时获得的积分倍数增加0.5倍
2. **爆牌惩罚**：爆牌时，根据要牌数量倍增扣除积分
3. **游戏结束**：当玩家积分低于5分时，游戏结束，需要点击「开始」按钮重置游戏

## 使用方法

1. **选择押注**：点击押注按钮选择押注金额（5分、10分、20分、50分、100分）
2. **开始游戏**：点击「开始」按钮开始新一局游戏
3. **游戏操作**：
   - 点击「要牌」获取一张新牌
   - 点击「停牌」结束你的回合，让庄家继续
4. **游戏结果**：游戏结束后，系统会显示胜负结果和积分变化
5. **重置游戏**：当积分不足5分时，点击「开始」按钮重置游戏并恢复初始积分

## 开发说明

### 文件结构

- `index.html`：包含游戏的完整HTML、CSS和JavaScript代码
- `README.md`：项目说明文档

### 主要类与方法

- **BlackjackGame**：游戏主类，管理整个游戏流程
- **initializeElements()**：初始化游戏界面元素
- **setupEventListeners()**：设置事件监听器
- **startGame()**：开始新游戏
- **resetGame()**：重置游戏状态
- **hit()**：玩家要牌
- **stand()**：玩家停牌
- **endGame()**：结束游戏并计算结果
